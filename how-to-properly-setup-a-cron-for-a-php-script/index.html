How to properly setup a cron for a php script

Jul 30, 2013

use / usr/bin/php /var/www/…/script.php

instead of

lynx - dump http://..../script.php

Reasons

1. The most important, the imposed execution time from apache is lift-offed so the chances for the script to stop from executing are smaller

2. No more apache overhead, starts faster, less memory consumtion for the server
